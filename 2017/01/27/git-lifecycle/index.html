git-lifecycle

2017-01-27
git

Lifecycle이란?

짧게 말하면 한국어로 생명주기이다.

안드로이드의 Activity 라이프사이클(물론 글쓴이는 네이티브 안드로이드를 해보지 않았다.)
React의 Component 라이프사이클, IOS의 View 라이프사이클 등등이 있다.

대체로 아래 그림과 같은 구조를 가지고 있다. 다만 cycle이라고 해서 꼭 순환되지는 않는 경우도 있다. (역주행 하는 경우도 있다.)


Git의 lifecycle

Git에도 이처럼 4가지의 Lifecycle을 가지고 있다.

  1. Untracked
  2. Unmodified
  3. Modified
  4. Staged

말로만 설명하면 어려우니 아래 그림을 참조해보자. 명령어와 같이 볼 수 있어서 더 이해가 쉬운 그림이다.
